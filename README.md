# 100マス計算（GIGAスクール構想対応版）

GIGAスクール構想で導入された端末（ChromebookやiPadなど）で、子どもたちが快適に計算練習に取り組めるように設計された「完全スタンドアロン」のデジタル100マス計算アプリです。

学校のネットワーク環境（フィルタリングや回線速度）に左右されず、ブラウザを開いた瞬間にサクサク動くことを最優先に開発されました。

## ✨ 主な機能

*   **💻 爆速・完全オフライン動作** 外部データベースや複雑なバックエンドサーバーを必要とせず、GitHub Pagesとブラウザの機能だけで完結して動作します。
    
*   **📝 3つの入力インターフェース**
    
    *   **AI手書き入力:** TensorFlow.jsを活用し、画面に直接数字を書いて入力できます（独自モデルをローカルで読み込み）。
        
    *   **ソフトウェアテンキー:** タッチパネルで押しやすい大型ボタンによる入力。
        
    *   **物理キーボード:** 十字キーとEnterキーによる高速なタイピング入力にも対応。
        
*   **⚡ 自動採点（すぐ判定）＆ 自動進行** 正解を入力した瞬間に「ピンポン！」と音が鳴り、自動で次のマスへ移動。間違えた場合は入力が自動リセットされ、思考のテンポを止めません。
    
*   **📊 成長が見える記録機能** ブラウザのLocalStorageを利用し、各計算モードの「ベストタイム」と「直近20回の記録」を自動的に保存・表示します。
    

## 🛠️ 技術スタック

*   **フロントエンド:** React 18 (CDN読み込み) / JSX
    
*   **スタイリング:** Tailwind CSS (CDN読み込み)
    
*   **アイコン:** Lucide React
    
*   **AI・機械学習:** TensorFlow.js (手書き数字認識 / `model.json` を使用)
    
*   **音声:** Web Audio API (外部音声ファイル不要の動的シンセサイザー)
    
*   **ホスティング:** GitHub Pages
    

## 📁 ディレクトリ構成

GitHub Pagesで公開するために必要なファイルは以下の4つのみです。

```
.
├── index.html            # アプリの土台（Reactや各種ライブラリの読み込み）
├── App.jsx               # アプリの全UIとロジック（Reactコンポーネント）
├── model.json            # AI手書き認識用のモデル定義ファイル
├── group1-shard1of1.bin  # AI手書き認識用の重みデータファイル
└── favicon.png           # アプリのアイコン画像
```

## 🚀 デプロイ（公開）方法

このアプリは、プログラミングの専門知識がなくてもGitHub Pagesを使って無料で公開できます。

1.  このリポジトリをフォーク（またはクローン）します。
    
2.  GitHubのリポジトリの `Settings` > `Pages` を開きます。
    
3.  `Build and deployment` の `Source` を `Deploy from a branch` に設定します。
    
4.  `Branch` を `main` (または master)、ディレクトリを `/(root)` に設定して `Save` を押します。
    
5.  1〜2分待つと、上部に公開されたURLが表示され、世界中からアクセスできるようになります。
    

## 💡 使い方（児童向け）

1.  アプリを開いたら、「計算（たし算・引き算・かけ算）」と「問題数（10〜100問）」を選びます。
    
2.  「スタート！」ボタンを押すとタイマーが動きます。
    
3.  空いているマスに答えを入力します。
    
    *   「設定」ボタンから、入力方法（手書き・ボタン）や、入力エリアの左右の配置を変更できます。
        
4.  全部のマスを埋めるとクリア！タイムが記録されます。
    
5.  右上の「グラフ」ボタンから、これまでの自分の成長（記録）を確認できます。
