# 🧮 100マス計算 (Online 100-Square Calculation)

GIGAスクール端末（Chromebook）やタブレットでの利用に最適化された、ブラウザで動く100マス計算アプリです。 単なる計算ドリルにとどまらず、**AI（TensorFlow.js）を用いた手書き入力**や、**Googleスプレッドシートへの自動成績記録**など、教育現場の実務を強力にサポートする機能を備えています。

## ✨ 主な特徴

*   **選べる3つの入力方法**:
    
    1.  **手書き入力**: AIが画面に書かれた数字を自動で読み取ります。
        
    2.  **ソフトウェアテンキー**: 画面上のボタンをタップして入力できます。
        
    3.  **キーボード**: 物理キーボードでのスピーディーな入力にも対応。
        
*   **教員の負担軽減（自動記録）**: 児童が計算を終えると、タイムや点数が自動的に連携されたGoogleスプレッドシートに記録されます（クラス全員の同時アクセスにも対応した排他制御済み）。
    
*   **児童に寄り添うUI**: 重要な漢字へのふりがな（ルビ）対応、正解・不正解時の分かりやすい音声フィードバック、直感的な設定画面。
    
*   **GIGAスクール標準対応**: Chromebookの画面サイズに最適化されたレスポンシブデザイン。
    

## 🛠 セットアップ方法 (GASとしてのデプロイ)

このアプリは **Google Apps Script (GAS)** のWebアプリとして動作します。以下の手順で設定を行ってください。

### Step 1: スプレッドシートの準備

1.  新規のGoogleスプレッドシートを作成します。（これが成績の保存先になります）
    
2.  メニューの `拡張機能` > `Apps Script` をクリックし、スクリプトエディタを開きます。
    

### Step 2: ファイルの配置

GitHubリポジトリ内のコードをコピーし、以下の構成でファイルを作成・貼り付けます。

*   `コード.gs` (または `Code.gs`)
    
*   `index.html`
    
*   `css.html`
    
*   `js.html`
    

### Step 3: アセットファイル（AIモデル・音声）の準備

このアプリは効果音やAIモデルをGoogleドライブから読み込みます。

1.  必要なアセット（`.json`, `.bin`, `.mp3`等）を自身のGoogleドライブにアップロードします。
    
2.  アップロードした各ファイルの共有設定を **「リンクを知っている全員（閲覧者）」** に変更します。
    
3.  各ファイルの **ファイルID**（URLの `d/` と `/view` の間の文字列）をメモします。
    
4.  `コード.gs` 内の `FILE_IDS` オブジェクトの各IDを、あなたがアップロードしたファイルのIDに書き換えます。
    

```
// コード.gs の以下の部分を書き換えてください
var FILE_IDS = {
  modelJson: 'あなたの_jsonファイルの_ID',
  modelBin: 'あなたの_binファイルの_ID',
  soundCorrect: 'あなたの_正解音の_ID',
  soundIncorrect: 'あなたの_不正解音の_ID',
  soundEnd: 'あなたの_終了音の_ID'
};
```

### Step 4: Webアプリとしてデプロイ

1.  エディタ右上の `デプロイ` > `新しいデプロイ` をクリック。
    
2.  種類の選択（歯車アイコン）から `ウェブアプリ` を選択。
    
3.  以下の設定にしてデプロイします。
    
    *   **実行するユーザー**: `自分` または `ウェブアプリにアクセスしているユーザー` (※組織の運用に合わせてください)
        
    *   **アクセスできるユーザー**: `全員` または `Google アカウントを持つ全員`
        
4.  承認が求められたら許可を進め、表示された **ウェブアプリのURL** を児童に共有します。
    

## 👨‍💻 カスタマイズについて

*   **デフォルト設定の変更**: `js.html` の `appSettings` オブジェクトを編集することで、初回起動時の「音の有無」「手書き入力のオンオフ」などのデフォルト状態を変更できます。
    
*   **テーマカラー**: `css.html` の `:root` セクションにある CSS変数を変更することで、学校のカラーに合わせたカスタマイズが可能です。
    

## 📂 ファイル構成

*   `code.gs`: バックエンド処理（スプレッドシートへの記録、ドライブからのデータ読み込み）
    
*   `index.html`: メインのHTML構造（Bootstrap 5使用）
    
*   `css.html`: スタイルシート
    
*   `js.html`: フロントエンドロジック（UI制御、AI推論、タイマー等）
